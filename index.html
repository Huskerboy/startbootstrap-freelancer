    Ride the Bus

    Ride the Bus originated as a simple drinking game, but through the use of python I developed it into a game that could be run on the command prompt. It follows a very simple set of rules that require to you correctly guess certain attributes of the next card. If you guess it right you move on to the next card in the sequence, if you guess it wrong you have to start over; there are four cards total.

    1. For the first card you must guess the color. So this gives you two option, either red or black.
    2. You must guess whether the second card's value is higher or lower than the value of the first. So if the first car is an Ace, then you will want to guess lower because Ace is high. If the second card is of the same value as the first, then you have to drink twice and restart.
    3. The third guess is whether you believe that the third cards value will be inside or outside of the first two cards values. So if it is a 6 and an 8, then you will want to guess outside. If the values of the third card matches that of one of the first two, then you must drink twice and restart.
    4. Your forth and final guess is to guess the suit of this card. Since it does not depend on the first 3 cards at all, the value can match that of the first 3 without penalty.
